# Stitch Alerta App

Aplicativo de segurança e alerta criminal desenvolvido com **React Native**, **Expo** e **NativeWind**.

## Funcionalidades Migradas
- **Relatório de Incidente**: Tela completa para reportar ocorrências com mapa e fotos.
- **Botão de Pânico (SOS)**: Botão de emergência com animação e envio de alerta.
- **Mapa de Segurança**: Visualização de áreas de risco e filtros por distância.
- **Perfil do Usuário**: Gamificação, níveis e gestão de assinatura.
- **Configurações**: Gerenciamento de notificações e privacidade.
- **Dashboard Admin**: Gráficos e estatísticas para administradores.

## Como Rodar

### Pré-requisitos
- Node.js instalado.
- Dispositivo Android ou Emulador.

### Passos
1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o projeto:
   ```bash
   npx expo start
   ```

3. Escaneie o QR Code com o app **Expo Go** no seu celular ou pressione `a` para abrir no emulador Android.

### Nota sobre Web
A versão web pode apresentar instabilidade devido à versões recentes do Expo SDK 54. Recomenda-se testar prioritariamente no ambiente **Android/iOS** para garantir a experiência nativa correta.
